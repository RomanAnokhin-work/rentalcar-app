# 🚗 RentalCar App

## 📌 Назва проєкту
**RentalCar App**

## 📝 Короткий опис
RentalCar App — це веб-застосунок для перегляду та оренди автомобілів.  
Користувачі можуть переглядати список авто, використовувати фільтри, додавати автомобілі в обране та переглядати детальну інформацію про кожну машину.

Проєкт реалізований як сучасний frontend-додаток з використанням сучасного стеку технологій.

🔗 Демо: https://rentalcar-app-one.vercel.app/

---

## ⚙️ Основні функції

- 📋 Перегляд списку автомобілів
- 🔍 Фільтрація авто (за брендом, ціною, пробігом тощо)
- ❤️ Додавання авто в "Обране"
- 📄 Пагінація списку
- 🖼️ Перегляд детальної інформації про автомобіль
- 📱 Адаптивний дизайн (під мобільні пристрої)
- 🚀 Оптимізована продуктивність

---

## 🛠️ Використані технології

- **Next.js**
- **React**
- **TypeScript**
- **CSS Modules**
- **Zustand** (керування станом)
